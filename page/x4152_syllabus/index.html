Course Description

Gain an understanding of the core concepts of data science illustrated through the use of the R language. Learn the data science lifecycle, roles, and fundamentals and build a solid foundation before diving deeper into the theory and practice of predictive analytics and programming in future courses.

Course Learning Objectives

When you have completed the course, you should be able to:

Understand the data science process and fundamental concepts
Perform exploratory data analysis and create data visualizations
Understand critical statistical concepts
Avoid common data science pitfalls
Understand predictive analytics from a high level
Navigate and use the R programming environment
Write basic R programs using the tidyverse
Collaborate and communicate results effectively

Intended Audience

This course is for students wanting a gentle, broad and practical introduction to the data science process and fundamental concepts, with lessons and examples illustrated through the use of the R programming language. This course is not meant for students who want a deep dive into programming or predictive modeling, but is meant for those looking for a high-level understanding of the data science field and a solid foundation of data science principles before pursuing further programming and modeling courses.

Course Work

This course consists of lectures once per week, for three hours, 6-7 assignments, a take home midterm and a take home final. All work is to be completed individually. We will set aside time during class for group work/discussions.

We will be using the R programming language for this course, so you will definitely need access to a computer with R installed. I will send detailed instructions on installing R on your machine. Laptops will be required during class. I will set aside time each session for “lab” activities so you can practice with R.

Prerequisites

The only prerequisite is an introductory statistics course. It will of course help if you have some programming or math background, but this course is taught at a high level.

Course Policies

Students are expected to turn in assignments by 6:15PM every Tuesday. Any later than this, and the assignment will be considered late and the grade reduced by 10% for each day late. For example, if the assignment is turned in after 6:15PM on Tuesday, the maximum grade is 90%. If it is turned in after 6:15PM on Wednesday, the maximum grade is 80%, and so on.

The midterm and final are take-home assignments and will be due by 6:15PM on the Tuesday after they are assigned. These will not be accepted late as we will be going over the solutions during class.
